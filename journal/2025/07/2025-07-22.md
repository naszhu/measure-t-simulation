### The following 2 change back from wayland to x11:

- 禁用 `ubuntu-wayland.desktop`：
    
```bash
sudo mv /usr/share/wayland-sessions/ubuntu-wayland.desktop \         /usr/share/wayland-sessions/ubuntu-wayland.desktop.disabled

```
    
- 禁用 `ubuntu.desktop`（它也是一个 Wayland 会话描述文件）：
    
    
```bash
sudo mv /usr/share/wayland-sessions/ubuntu.desktop \         /usr/share/wayland-sessions/ubuntu.desktop.disabled
```
