- **[[sample]] vs. [[event]]**:
    
    - **Sample**: An element ω∈Ω (a single outcome).
    
    - **Event**: A measurable set A∈F (collection of outcomes).  something to which you can assign a probability via P(A)P(A)P(A).
    - **Example**: For Ω={1,2,3,4,5,6} (die roll), ω=3 is a [[sample]]; A={2,4,6} (even numbers) is an [[event]].
