â— To add Anthropic/Claude domains to Clash Verge rules:

  1. Open Clash Verge
  2. Go to "Profiles" tab
  3. Click the edit button (pencil icon) next to your active profile
  4. Find the "rules:" section and add these lines at the top:

  rules:
```
DOMAIN-SUFFIX,anthropic.com,PROXY
```
```
DOMAIN-SUFFIX,claude.ai,PROXY
```
```
DOMAIN-KEYWORD,claude,PROXY
```
```
DOMAIN-KEYWORD,anthropic,PROXY
```


```
prepend: []

append:

- 'DOMAIN,DOMAIN-SUFFIX,anthropic.com,PROXY,DIRECT'

- 'DOMAIN-SUFFIX,claude.ai,PROXY'

- 'DOMAIN-KEYWORD,claude,PROXY'

- 'DOMAIN-KEYWORD,anthropic,PROXY'

delete: []
```



  5. Save and switch to "Rule" mode

  global mode.


# Auto-detect mouse
```bash
  # Create the script
  nano ~/bluetooth-mouse-connect.sh

  Add this content:
  #!/bin/bash
  sleep 10
  bluetoothctl connect F6:AA:20:68:98:81

  # Make executable
  chmod +x ~/bluetooth-mouse-connect.sh

  Add to startup:

  # Add to your shell startup file
  echo "~/bluetooth-mouse-connect.sh &" >> ~/.bashrc


```

```bash
  # Add to system startup applications:

%%   1. Open "Startup Applications" (search in your app menu)
  1. Click "Add"
  2. Fill in:
    - Name: Bluetooth Mouse Auto-connect
    - Command: /home/lea/bluetooth-mouse-connect.sh
    - Comment: Auto-connect MX Anywhere 3
  3. Save %%

  OR use this command:
  # Add to desktop autostart
  mkdir -p ~/.config/autostart
  cat > ~/.config/autostart/bluetooth-mouse.desktop << EOF
  [Desktop Entry]
  Type=Application
  Name=Bluetooth Mouse Auto-connect
  Exec=/home/lea/bluetooth-mouse-connect.sh
  Hidden=false
  NoDisplay=false
  X-GNOME-Autostart-enabled=true
  EOF

```
